Мощный модульный инструментарий для реверс-инжиниринга, перевода и модификации игры **Mytran Wars** (PlayStation Portable). Создан для полной русификации игры, включая перерисовку шрифтов, перевод текстов и распаковку архивов.

## 🚀 Основные возможности

### 🎨 Font Atlas Studio & Generator (Ядро проекта)
Гордость данного тулбокса — продвинутый визуальный редактор и авто-генератор шрифтов, встроенных в `EBOOT.BIN`. Игровой движок использует жестко заданные рамки (`Advance × Height`). Наш алгоритм решает проблему нехватки места и "прыгающих" букв:
* **Global Tetris Repack:** Алгоритм стирает оригинальный атлас и переупаковывает все символы (латиницу, знаки, кириллицу) с нуля, вычисляя оптимальные пустые зоны (Bin Packing).
* **Smart Baseline Sync:** Скрипт отсекает мусорные полупрозрачные пиксели (Ghost Pixel Culling) и математически привязывает новые буквы к оригинальной базовой линии. Текст больше не "прыгает".
* **Live Preview:** Встроенный симулятор PSP-экрана с отрисовкой опорных линий и рамок коллизий игрового движка.
* **Кастомная латиница:** Возможность полностью перерисовать оригинальный английский алфавит из любого TTF-шрифта.
* Поддержка "Синтетического жирного" (Fake Bold) и ручного сглаживания альфа-канала.

### 📦 Распаковщик ресурсов
* **.PPK Архивы:** Чистая распаковка архивов с созданием манифеста (`_filelist.txt`) и запаковка обратно без мусорных файлов.
* **.PTEX Текстуры:** Конвертация "свиззленных" (Swizzled) PSP-текстур в `.png` и обратно с сохранением палитры и альфа-канала.
* **.SCR Экраны:** Работа с RAW RGBA экранами загрузки (480x272).

### 📝 Текстовый движок
Модуль для комфортного перевода игры. Извлекает бинарные текстовые блоки в человекочитаемый `.txt` и собирает их обратно с авто-расчетом байтовых длин и перекодировкой в `Windows-1251`:
* Поддержка файлов целей миссий (`*.loc`).
* Поддержка диалогов и катсцен (`mission_*.bin`).
* Поддержка файлов интерфейса (`menu.bin`, словари UI).

## 🛠 Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/majerock/Mytran_Wars_toolbox.git
   ```
2. Установите зависимости (требуется только Pillow для работы с графикой):
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите оболочку:
   ```bash
   python main.py
   ```
